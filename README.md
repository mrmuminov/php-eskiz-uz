PHP Eskiz.uz
============

PHP Eskiz.uz SMS Gateway package

## Installation

```bash
composer require mrmuminov/php-eskiz-uz "^2.0.0"
```

or add the following to your `composer.json` file:

```json
{
    "require": {
        "mrmuminov/php-eskiz-uz": "^2.0.0"
    }
}
```
